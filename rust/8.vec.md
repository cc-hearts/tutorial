vector 由标准库提供 可以存储多个值 只能存储相同类型的数据 在内存中连续存放(类似于连续的数组)

```rust
fn main() {
    println!("Hello, world!");
    // vector 由标准库提供 可以存储多个值 只能存储相同类型的数据 在内存中连续存放
    // Vec<T>
    let mut v: Vec<i32> = Vec::new();

    // vec! 宏 可以使用初始值创建Vector

    // 创建
    v.push(1);
    v.push(2);
    v.push(3);
    v.push(4);

    // 获取
    // 1.索引 超出边界 会出现Panic
    // 2.通过get方法
    let third: &i32 = &v[2];
    print!("{}", third);
    match v.get(2) {
        Some(third) => println!("{}", third),
        None => println!("None"),
    }

    vec_test();

    vec_enum();
}

fn vec_test() {
    let mut test = vec![1, 2, 3];

    for i in &mut test {
        // 对地址值解引用
        *i += 50;
        println!("{}", i);
    }
}

fn vec_enum() {
    // enum 变体可以存放多种数据类型
    #[derive(Debug)]
    enum SPreadSheetCell {
        Int(i32),
        Float(f64),
        Text(String),
    }
    let row = vec![
        SPreadSheetCell::Int(1),
        SPreadSheetCell::Float(2.0),
        SPreadSheetCell::Text(String::from("hello")),
    ];
    // for循环的方式获取元素
    for cell in &row {
        match cell {
            SPreadSheetCell::Int(i) => println!("{}", i),
            SPreadSheetCell::Float(f) => println!("{}", f),
            SPreadSheetCell::Text(t) => println!("{}", t),
        }
    }
    let u = row.get(0);
    match u {
        Some(i) => match i {
            SPreadSheetCell::Int(i) => println!("{}", i),
            SPreadSheetCell::Float(f) => println!("{}", f),
            SPreadSheetCell::Text(t) => println!("{}", t),
        },
        None => println!("None"),
    }
}

```

## 切片类型

```rust
// 比较大小 返回切片中最大的值
// &[T] 表示一个切片 T 是一个切片类型
fn compare_largest<T>(list: &[T]) -> &T
// 实现了 PartialOrd 的 trait 才可以比较大小
where
    T: std::cmp::PartialOrd,
{
    let mut largest = &list[0];

    // list 是 &[T] 类型。 如果直接使用 item in list  item的类型会是 &T 可以使用& 解构
    for item in list {
        if item > largest {
            largest = &item
        }
    }
    return largest;
}
```

---
title: border-radius
---

`border-radius` 如果用了 `/ ` 分开的话 表示的是一个椭圆 前面的是横轴 后面的是竖轴。

```css
#root {
  width: 200px;
  height: 200px;
  border-radius: 20px / 50px;
  background-color: red;
}
```

```html
<div id="root"></div>
```
